# Trading Ledger

This folder is intentionally simple. Use `group_stage_round_1/` for the first round of group-stage matches.

Each match has one Markdown file containing compact model input, prediction, bet tracking fields, and final result / profit fields.
`group_stage_round_1/summary.csv` is the one-file overview. After each match settles, update the match file and the matching summary row, including `running_average_pnl_usd`.
